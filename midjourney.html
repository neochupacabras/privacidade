Política de Privacidade

Última atualização: 24 de maio de 2025

A extensão Midjourney Auto Prompt foi desenvolvida exclusivamente para automatizar o envio de prompts para a interface do site Midjourney Web.

Quais dados são coletados?

Nenhum dado pessoal é coletado. Todas as configurações, histórico de prompts e preferências são armazenados localmente no navegador do usuário.

Compartilhamento de dados

Esta extensão não compartilha dados com terceiros. Toda a lógica de funcionamento é local e independente de servidores externos.

Permissões utilizadas

A extensão solicita permissões para acessar a aba do Midjourney com o único objetivo de injetar os prompts no campo de texto e clicar no botão de envio de forma automatizada, conforme configurado pelo usuário.
