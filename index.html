Política de Privacidade

Última atualização: 24 de maio de 2025

A extensão Leonardo AI Auto Prompt foi criada com o único propósito de automatizar o envio de prompts para o site Leonardo AI. Esta extensão não coleta, armazena ou compartilha dados pessoais dos usuários.

Quais dados são coletados?

Nenhum dado pessoal é coletado. Todas as configurações e históricos são armazenados localmente no navegador do usuário, sem qualquer comunicação com servidores externos.

Compartilhamento de dados

Não há compartilhamento de dados com terceiros. A extensão opera inteiramente dentro do navegador.

Permissões usadas

A extensão solicita permissões para interagir com o site do Leonardo AI com o objetivo exclusivo de automatizar o envio de prompts. Nenhuma outra funcionalidade é implementada.
